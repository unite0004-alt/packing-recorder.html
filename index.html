開啟自動上傳到 Google Drive
Google Drive 設定教學（一次性）
  1. 到 console.cloud.google.com 建立一個專案。
  2. 左側「API 和服務 → 程式庫」搜尋 Google Drive API 並啟用。
  3. 「OAuth 同意畫面」選 外部，填基本資料，把自己的 Google 帳號加進測試使用者。
  4. 「憑證 → 建立憑證 → OAuth 用戶端 ID」，類型選 網頁應用程式。
  5. 在已授權的 JavaScript 來源貼上本頁網址（見最下方），不要結尾斜線。
  6. 把產生的 用戶端 ID 貼到上面欄位，按「連結 Google」。

貼上你自己的 App key，影片只會傳到你的 Dropbox。下方有設定教學。

開啟自動上傳到 Dropbox
Dropbox 設定教學（一次性）
  1. 到 dropbox.com/developers/apps → Create app。
  2. 選 Scoped access → App folder（只會建立一個專屬資料夾，最安全）。
  3. 在 Permissions 勾選 files.content.write 與 files.content.read，按 Submit。
  4. 在 Settings → OAuth 2 → Redirect URIs 加入本頁網址（見最下方）。
  5. 把 App key 貼到上面欄位，按「連結 Dropbox」並登入授權。
